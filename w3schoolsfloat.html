CSS float an element can be pushed to the left or right, allowing other elements to wrap around it. Float is often used for images, but also useful working with layouts.

Elements are floated horizontally meaning an element floats left or right only. Not up or down. A floated element moves as far to the left or right as it can. Usually all the way left or right of the containing element. The elements after the floating element flows around it. The elements before the floating element is not affected.

If an image is floated to the right, a following text flows around it, to the left.
