A block element is an element that takes up the full width available and has a line break before and after it. Examples are h1, p, and div.

Center align using the margin property. block elements can be center-aligned by setting the margin-left and margin-right to "auto". A shortcut is margin: auto; Setting the left and right margins to auto specifies they split the available margin equally. The result is a centered element below. The width 70% can be specified in px. 100% is useless because the block element stretches from left to right.
