An outline is a line that is drawn around elements outside the outlines to make the element stand out. The outline property is different from the border property. The outline is not part of an element's dimensions; the element's total width and height is not affected by the width of the outline. A margin 25px was assigned to p element because outline doesn't affect the element's dimensions. If there was no margin, groove and ridge outlines overlapp.
